# Telegram Air Alert Bot 🚨

Бот для Telegram, який автоматично надсилає повідомлення про повітряну тривогу та відбій у ваш канал або чат.

---

## 🚀 Особливості

 - Відправка тривог та відбоїв у Telegram-канал
 - HTML-форматування та клікабельні посилання
 - Антиспам — не дублює одні й ті ж статуси
 - Працює навіть якщо Render Free Web Service спав
 - Логування рестарту у Render (щоб бачити, коли бот прокинувся)
 - Простий Express-сервер для пінгу (UptimeRobot)

---

## ⚡ Встановлення

1. Клонуйте репозиторій:

```bash
git clone https://github.com/SerhiiRomanenko/telegram-air-alert-bot.git
cd telegram-air-alert-bot

```

2. Встановіть залежності:

```bash
npm install
```

3. 🔧 Запуск

```bash
node index.js
```
